Mes tous premiers paragraphes stylé avec javascript

JavaScript est un langage de script très populaire, utilisé notamment afin d’implémenter des traitements dans les pages Web.
Tout au long de cet ouvrage, nous nous efforçons de détailler les différents concepts de ce langage ainsi que les mécanismes à
mettre en œuvre afin de l’utiliser dans ce type de pages Le langage JavaScript permet de réaliser des traitements évolués dans
les pages Web en offrant la possibilité d’utiliser les différents standards Web supportés
